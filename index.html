API Digest

20-21 February 2021


API Digest is a competition to test the knowledge and expertise of students in API based applications. Participants will be given themes and APIs to create their project and they are required to finish their projects along with the presentation in thirty hours. The API application can be demonstrated using either web or mobile applications. Participants can also use APIs not provided by the organizers. Each team will be of strictly 2 members will  have a leader who will be the point of contact to the organizers.
